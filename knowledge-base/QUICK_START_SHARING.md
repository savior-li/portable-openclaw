## 💡 30秒演示

### 演示1: 自动知识提取
```python
from knowledge_base import learn_from_instruction

# 用户说人话
learn_from_instruction("所有提交前必须运行测试")

# AI自动提取规则
# ✅ category: deny (禁止)
# ✅ priority: high (高优先级)
# ✅ tags: ["测试", "工作流"]
# ✅ confidence: 0.8
```

### 演示2: 智能冲突解决
```python
# 发现相似规则
规则1: "不要删除文件" (avoid, medium)
规则2: "禁止删除文件" (deny, high)

# 自动合并 → "禁止删除文件" (deny, high)
```

### 演示3: 知识衰减
```python
# 自动追踪规则使用情况
30天未使用 → 效果分数 × 0.9
60天未使用 → 效果分数 × 0.81
# 保持知识库新鲜度
```
